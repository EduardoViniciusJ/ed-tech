## REFACTOR015 - Padronizar IDs das Páginas Home do Portal

**Update Set**: `EDTECH_05_PORTAL_V2_REFACTOR010_PADRONIZAR_IDS_HOMES`  
**Parent**: `EDTECH_05_PORTAL_V2`

**O que precisa**:  
Revisar e padronizar os IDs e links das páginas principais do Portal EdTech.

**Padrão obrigatório**:

- `home_aluno`
- `home_gestor`
- `home_financeiro`
- `home_suporte`
- `home_secretaria`


Depois atualizar o widget de redirecionamento EdTech - Redirecionar Home por Perfil  o server script para esse Código aqui:  

(function() {
    data.redirectUrl = '';
    data.message = '';

    if (gs.getUser().isMemberOf('Professores')) {
        data.redirectUrl = '?id=home_professor';
        return;
    }

    if (gs.getUser().isMemberOf('Alunos')) {
        data.redirectUrl = '?id=home_aluno';
        return;
    }

    if (gs.getUser().isMemberOf('Suporte')) {
        data.redirectUrl = '?id=home_suporte';
        return;
    }

    if (gs.getUser().isMemberOf('Financeiro')) {
        data.redirectUrl = '?id=home_financeiro';
        return;
    }

    if (gs.getUser().isMemberOf('Secretaria')) {
        data.redirectUrl = '?id=home_secretaria';
        return;
    }

    if (gs.getUser().isMemberOf('Gestores Acadêmicos')) {
        data.redirectUrl = '?id=home_gestor';
        return;
    }

    data.message =
        'Seu perfil ainda não está configurado para acessar este portal.';
})();

**Critérios de aceite**:

- Todas as páginas seguem os IDs definidos.
- Navegação e redirecionamento por perfil funcionam.
- Nenhum link apresenta página não encontrada.
- Os links funcionam dentro do Portal EdTech.
