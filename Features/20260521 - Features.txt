## STRY0012 - Criar Variable Set De Suporte Técnico

**Prioridade**: Alta  
**Update Set**: EDTECH_02_SUPORTE_STRY0015_VS_SUPORTE_TECNICO  
**Parent**: EDTECH_02_SUPORTE

**Persona**:  
Aluno ou professor que precisa relatar problemas técnicos no AVA, Moodle, portal ou envio de atividades, informando detalhes suficientes para que o Suporte consiga entender e resolver o problema rapidamente.

**O que precisa**:  
Criar um Variable Set específico para chamados de Suporte Técnico, com perguntas voltadas aos problemas core que estão no arquivo de TCC: aula que não abre, vídeo que não carrega, erro no envio de atividade e dificuldade de acesso ao portal/AVA.

**Critérios de aceite**:

- Variable Set **Detalhes do Problema Técnico** criado.
- Campo **Sistema afetado** criado com opções: AVA/Moodle, Portal do Aluno, Vídeo/Aula, Envio de Atividade, Outro.
- Campo **Tipo de problema** criado com opções: Aula não abre, Vídeo não carrega, Erro no envio da atividade, Problema de acesso, Outro.
- Campo **URL ou caminho da tela com erro** criado.
- Campo **Dispositivo utilizado** criado.
- Campo **Navegador utilizado** criado.
- Campo **Mensagem de erro apresentada** criado.
- Campo **Data e horário aproximado do problema** criado.
- Campo **Anexo de evidência/print** disponível.

## STRY0015 - Ajustar Record Producer De Suporte Técnico

**Prioridade**: Alta  
**Update Set**: EDTECH_02_SUPORTE_STRY0018_RP_SUPORTE_CORE  
**Parent**: EDTECH_02_SUPORTE

**Persona**:  
Aluno ou professor que precisa registrar um problema técnico no portal, AVA ou Moodle e acompanhar a resolução sem depender de e-mail ou atendimento informal.

**O que precisa**:  
Atualizar o Record Producer **Relatar Problema com AVA/Moodle** para usar os Variable Sets comuns e o Variable Set específico de Suporte Técnico. O formulário deve cobrir os principais problemas relatados no PDF sem criar vários formulários separados para cada tipo de erro.

**Critérios de aceite**:

- Record Producer **Relatar Problema com AVA** atualizado.
- Formulário usa os Variable Sets comuns.
- Formulário usa o Variable Set **Detalhes do Problema Técnico**.
- O campo **Tipo de problema** permite selecionar aula que não abre, vídeo que não carrega, erro no envio de atividade ou problema de acesso.
- Ao enviar, um registro é criado em Incident [incident].
- O chamado é direcionado para o grupo **Suporte**.
- A linguagem do formulário é clara para aluno e professor.
